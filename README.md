# peviitor_core
here is the core of peviitor project
mostly it's about data and the quality of data but it's also about getting the data you are searching for using full-text indexed search engine
BFF API, UI, scrapers, manual data validator will be considered as plugins for peviitor core project
security and procedures related to ways of working will be part of the project
how to connect and how to use it to  be captured in documentation
all pull requests will be documented
project is OPEN SOURCE
each data and type of data will be fully documented
peviitor core is not closed project but an extensible one
performance, reliability, recovery from desaster, scalability and validity are the most valuable benefits this project is delivering
